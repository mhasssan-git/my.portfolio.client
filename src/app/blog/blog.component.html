AWS Code Pipeline
Building CI/CD pipeline for angular application
AWS CodePipeline is a fully managed continuous delivery service provided by Amazon Web Services (AWS). It automates the process of releasing and deploying applications and infrastructure changes through various stages, from source code to production.By using AWS CodePipeline, development teams can implement a continuous delivery (CD) or continuous integration (CI) approach to software development, allowing for faster and more reliable releases of applications and infrastructure changes. Key features of AWS CodePipeline include: Pipeline Workflow,Integration with Various AWS Services,Source Control Integration, Artifact Management, Automated Testing, Customizable and Extensible,Visualization and Monitoring and scalability.
